Endangered Species Website

One of the first websites I ever created was for this project. We were required to put all of the research we had done on the endangered species my group had chosen, which was the Galapagos Penguin, into a website format. We started by producing photoshop mockups of what the end website would somewhat resemble (better looking of course). Then I brought our mockups to life by using Brackets to code the website. I had to constantly keep checking how it looked in the browser so I could fix its appearance if needed right away. Digital Ethics

In this project, our class had to work together to produce a website educating people about digital ethics and why they are so important. My group was given the topic of teen driving and the use of technology. My role was to explain the consequences of using technology while driving. I kept on having to write and rewrite code to try and conform to the rest of the website. For instance, I had to completely change the background to look like the rest of the website. I wrote the code for this in Brackets and resized the images using Photoshop. I can use this as a base to help me build up communication skills with other people within a group.
